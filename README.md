# 🛒 ShopSphere

## 🚀 Features
- Product Listing (API based)
- Product Details Page
- Search & Category Filter
- Add to Cart
- Cart Management (Update / Remove)
- Total Price Calculation
- LocalStorage Persistence
- Authentication + Protected Routes
- Toast Notifications

## 🛠️ Tech Stack
- React.js (Vite)
- Tailwind CSS
- Context API
- Axios

## 📦 Setup
```bash
npm install
npm run dev
